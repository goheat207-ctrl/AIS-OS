# Research Project Template

## Folder Structure

```
ACTIVE_PROJECTS\[project-name]\
├── CLAUDE.md              ← Fill from _template_project_claude.md
├── brief.md               ← Research question and scope
├── sources\
│   └── [source_name].md   ← Raw reference material
├── notes\
│   └── [topic].md         ← Working notes and synthesis
├── synthesis\
│   └── [output].md        ← Distilled findings
└── outputs\
    └── [deliverable].md   ← Final research output
```

---

## CLAUDE.md Additions for Research Projects

```
## Research Context

Question: [the core research question]
Scope: [what is in and out of scope]
Output Type: [memo / report / briefing / database entry]
Audience: [who will use this research]

- D:\AI_OS\THINKING_MODELS\thinking_frameworks.md
- D:\AI_OS\THINKING_MODELS\decision_templates.md [if decisions needed]
```

---

## Session Startup (Research)

1. Read project CLAUDE.md
2. Read research_workflow.md
3. Read thinking_frameworks.md
4. Load only the sources relevant to today's session

---

## Research Output Standards

Source notes:
- Named: `[author_or_source]_[topic].md`
- Contains: key claims, source URL or reference, credibility assessment

Synthesis notes:
- Named: `[topic]_synthesis.md`
- Contains: patterns across sources, conflicts, confidence level

Final output:
- Named: `YYYY-MM-DD_[topic]_[type].md`
- Contains: conclusion first, evidence second, sources last
