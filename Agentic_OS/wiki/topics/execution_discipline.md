# Execution Discipline

Profitable trading comes from eliminating execution errors. Not from better strategy or better stock selection.

> SMB Capital experiment: 20 stuck traders. Fixed 5 execution mistakes (not strategies). 16 of 20 back on track in 60 days.

---

## The 5 Execution Mistakes (SMB Capital)

### 1. Holding runners past 10:30 AM

73% of first-hour momentum runners made their HOD before 10:30.
89% before 11:00. Only 11% made new highs after 11:00.

**Fix:** 10:25 AM alarm on every first-hour momentum trade. Begin scaling out. Non-negotiable.

---

### 2. Adding to losing positions

Three forms — all the same mistake:
- **Averaging down** — buying more as price drops
- **Adding without EV expanding** — adding when the trade hasn't confirmed
- **Re-entering the same bad trade** — same idea, same lack of confirmation, just more hope

**The test before every add:**
> "What new information improved this trade? Would I take this fresh right here?"

If NO → the add is illegal.

> "Elite traders don't add because they want to be right. They add because the market proved them right."

---

### 3. Trading without tape confirmation

Chart-only entry group: 54% win rate  
Tape-confirmation group: 68% win rate — same setups, big gap

**4 tape signals — need 3 of 4 to confirm entry:**
1. Offer size thinning (big seller getting absorbed)
2. Sweep orders (multiple levels cleared — institutional buying)
3. Bids stacking on pullbacks (supportive buying)
4. Print speed accelerating (urgency increasing)

1 or 0 signals = wait or don't size up.

---

### 4. Ignoring the 3 red bar momentum shift

After 5+ consecutive green bars, 3 consecutive red bars = momentum shift signal.

**Rule:**
- Draw line at high of third red bar
- If next bar immediately takes it out → can hold
- If it does NOT → reduce size immediately (60%+ of remaining profits will be given back if you hold through this)

---

### 5. Not taking profits at resistance

First test of resistance holds 68% of the time.

**Rule:** At clear resistance (prior high, whole number, VWAP):
- Offer 50% of position 10 cents before the level
- If it breaks through: re-add with remaining half
- If it rejects: profits locked

---

## How to Fix

Fix one mistake at a time. Not all five at once.

1. Pick the mistake you KNOW you're making
2. Apply one rule for 60 trades minimum
3. Track the result
4. Add a second mistake to fix only after the first is consistent

Each mistake fixed is worth $300-$500/month individually.

---

## The Deeper Point

Most traders think they need:
- A better strategy
- Better stock selection
- Faster execution

Those are wrong diagnoses. The real problem is execution consistency.

Same strategies + same stocks + eliminated execution errors = profitable trader.

## Blaine's Honest Self-Assessment

Which of these 5 are currently costing the most? Use the journal to track:
- [ ] Holding past 10:30
- [ ] Averaging down / re-entering same bad trade
- [ ] Skipping tape confirmation
- [ ] Holding through 3-red-bar signal
- [ ] Holding into resistance without partial exit

Start with the one that shows up most in the journal.
